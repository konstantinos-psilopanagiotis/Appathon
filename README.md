Θα φτιαχτεί μια δικτυακή εφαρμογή η οποία βρίσκει τα 10 (max) πιο συχνά φάρμακα που έχουν χορηγηθεί (element: intervention) για την αντιμετώπιση μιας συγκεκριμένης ασθένειας. Το input θα είναι το όνομα της ασθένειας το οποίο θα το ορίζει ο χρήστης μέσω της web σελίδας και το output θα είναι το 10 (max) πιο συχνά φάρμακα (για την ακρίβεια, φαρμακευτικές ουσίες) καθώς και η συχνότητα χρήσης τους.

Για τις ανάγκες τις εφαρμογής αυτής θα χρησιμοποιηθούν οι διαθέσιμες κλινικές δοκιμές (XML αρχεία) από το https://clinicaltrials.gov/ ώστε αρχικά να φιλτράρονται οι κλινικές μελέτες με βάση την ασθένεια και έπειτα να βρίσκονται τα “top” φάρμακα.
